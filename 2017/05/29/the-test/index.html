中考无非是买一张演出门票

发表于 2017-05-29 分类于 随便说说 阅读次数： Valine：
本文字数： 822 阅读时长 ≈ 1 分钟

外甥女过些天就要中考了，端午节一起在外婆家吃饭，中考就成了讨论热点。

外甥女在本地最好的初中就读，不过近几次考试不是很理想，因此离最好的高中有点距离，但是应该也能上比较好的高中，说不定也能上那所最好的高中，毕竟考试还是有一定的偶然性。

小姑娘性格挺好的，也懂事。以前还有点小胖，这次看到瘦了很多。说是前些天还有点小焦虑，今天倒是还好。

从策略上来说，这么些天如果要努力，补自己的短板应该会有效点。毕竟从70分到80分，总比从90到100分容易点。但是我更想说的是从自己一生的角度来看，这样的一次考试应该怎样从心理上来对待它。

从小到大，我们要经历各种各样的升学考试，这些考试虽然是为了选拔人才，但是实际上并不总跟个人成才密切相关。当然也不可否认，在这些考试中胜出的人，在以后的人生博弈中确实会取得优势地位，但是社会生活中成功的人肯定要比考试成功的人多，而考试成功的人也不一定在成人世界里成功。

退一步说，这也不是一个不成功便成仁的世界，从人生的质量来说，幸福感可能比成功更重要。成功是很多人都渴望的，但是它不希望每个人都得到它，所以成功还跟机会有关，大多数的人在成功面前都只能说“尽人事，听天命”。但是幸福的感觉其实只要一个人运气不要太差，它都是个人可以把握的。一般来说脾气好、心态平和的人更容易有幸福感。外甥女拥有幸福的能力，这是从她出生到现在我一直能在她身上感受到的特点。所以她只要一直保持这种能力，本身又有积极向上的态度。从长远的角度，她能幸福，她能成功。

我更愿意把这么一次考试看作人生中为了看一次演出而去买门票。有能力的买一张贵一点的，能力弱点买一张便宜点的，买贵的可能提供的服务好点，位置好点，但是我们很难说贵的票一定就好，便宜的不好。人在小时候可能更看重成绩，但是到了大点的时候就看他是否积极，是否坚持。只有经历过了挫折的人才会在困难面前积极，坚持。所以，偶尔买一张差点的票，体验一下人生也不一定是坏事。

当然，无论如何还是要祝外甥女中考成功！
