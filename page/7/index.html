django使用思路

发表于 2015-02-18 更新于 2015-03-03 分类于 技术学习 Valine：

1.创建项目

django-admin.py startproject djangoweb

2.创建view

添加view.py文件，在里面添加代码:

from django.http import HttpResponse    

def hello(request):
    return HttpResponse("<h1>hello world!</h1>")

3.配置url

修改urls.py，假设可以用http://服务器地址/hello的方式来访问刚才定义的home这个视图，可以如下设置
from djangoweb.view import hello
urlpatterns = patterns(‘’, url(r’^hello/$’, hello, name=’hello’),)

这里的尖号（^）和美元符号（$）都是正则表达式符号，分别表示字符串的开头和结尾。

这是最简单的django项目，如果复杂的话就需要定义模板、视图，下次再说。

重拾django

发表于 2015-02-18 分类于 技术学习 Valine：

最近身体不舒服，做不了正经事，又重新摆弄起来django。上次使用django已经是七八年前的事了吧，那时候虽然早就熟练掌握asp.net，asp.net的使用已经变成了纯粹的体力劳动，所以当听说django的各种神奇，特别是能够自动生成管理界面的时候就心动不已（当时还不了解ror），何况对python语言也早就想用一用。不过那次也就就这django book玩了一遍，后来工作重心的转变也没继续玩。

这次继续摆弄也是托visual studio的福，因为ptvs的出现，使django开发的ide变得很熟悉，原来没有继续用django或python的一个很大的原因也在于ide的选择出现了困难，虽然用过很多eclipse，pycharm，eric，网上只有有人介绍过的差不多都试过，没有找到趁手的。现在能用visual studio那就不妨再用一下，所谓万事开头难，经历过n多的学习语言的经历，我知道到了一定程度后ide确实不再成为问题，但是一开始的时候好的ide确实给学习带来不少好处。

首先安装ptvs，可以在这里下载，安装一般不会有问题，它会自动找到本机安装的python。

开始使用，第一次创建django项目，它会去下载一些必要的组件，所以时间会有点长，一般来说只要你机器的python版本合适的话，就没问题了。我第一次创建就出问题了，后来发现是我机器的python版本太低所致，去官网下载最新的python后就ok了。

健康是福

发表于 2015-02-17 分类于 随便说说 Valine：

现代人生活好了，越来越关注自身的健康问题。君不见各种社交媒体中，特别是最近流行的微信中不是心灵鸡汤，就是养生秘笈，这些东西的流行至少说明大家对健康的关注。

我大概爹娘给我打的基础比较好，一直以来很少生病，所以以前很少为健康发愁。但是也有那么几次，对健康也是几近渴求。

在广州读硕士的时候有一次腰椎间盘突出，每每半夜痛醒，夜不能寐找小说看，最后找到了《相约星期二》居然就一口气看完，虽然依然痛，但是第一次感觉健康的珍贵，真心觉得缺什么也不要缺健康。

第二次是一不小心一个脚趾骨折，从没有没这么受伤过，所以当时也是很忧伤，极度怀念身体健康的时候。

最近也有两次，睡下去挺好的人，醒来的时候天旋地转，虽然不是痛，但是这种难受的感觉我宁愿痛。所以，又开始怀念健康了。

keep fit，真的太要紧了。

各种短视

发表于 2015-02-07 分类于 随便说说 Valine：

小刚导演说综艺电影是电影行业的自杀，还是有点道理的。不过商业社会各种存在都是合理的，关键是我们的公众需要一定的辨别力。拿研究来说，有些人三五天就写成一篇文章，有些人要一年半载，两者有不同的价值所在，前者多半是普及知识用的，而后者则是专业的知识创造，关键是要能区分这两篇文章背后所蕴含的学术意义，学术圈里实际上还是有自己的判断的，可惜的是现在管理学术圈的不一定懂学术，这也造成了数字学术，因为一个人学术再不济，能当领导数字还是看得懂的，数字多的就是好的。

电影业的问题也类似，因为电影本身就是大众货，要雅俗共赏，所以观众的鉴赏力很重要，可惜的是现在的大众传媒鉴赏口味都是从娃娃抓起的，看着儿子在追《奔跑吧，兄弟》我也偶尔陪他看一看。看着那些所谓的明星忸怩作态，矫揉造作我是觉得恶心的，设计的活动与情节也比较动物化。不过听说有人看这剧笑炸了肺，我想想还是让儿子看看吧，以免长大后笑点太低，也重蹈覆辙。所以，其实很难阻挡这类综艺电影的发展的，有点知识有点文化的人其实不用理会这些“黑”与“蜜”，做好自己最重要。想想我看的电影很少，基本上不是武侠就是香港的娱乐片，但这些经历似乎没有影响我做严肃的思考，而且我觉得自己是不是太严肃了点。
