给我上学的孩子

发表于 2013-09-01 更新于 2017-03-06 分类于 家庭生活 阅读次数： Valine：
本文字数： 735 阅读时长 ≈ 1 分钟

美国中部时间9月1日下午2点半，也是北京时间9月2日凌晨3点半，再过4个小时木木就要第一次进入他的小学了。远在他乡没法亲自陪他体验这人生的头一遭，就写点文字来迎接这一刻吧。

2003年在“一塌糊涂”上第一次看到《给我未来的孩子》，那时候你还不知道在哪里，但是做一个踏实的理想主义者却成为了我对你的期望。今天，你将第一次踏入学校，这也是你进入社会的预演，我的期望不变。

首先，我希望你是个独立的人。在过去的日子里，你也许没有机会独立，但是上学了，你在远离家人的时候，一定要学会做自己的事情，身体的劳动可以带来心灵的成长，你要多做事，首先从做自己的事开始，摆脱依赖的心理，从心理上独立起来。

其次，我希望你是一个勇敢的人。在过去的这些年中我知道你是个善良的人，情绪容易被外界所牵动，但是这不用难堪，流泪不代表什么，重要的是勇敢面对。遇到困难要积极解决，方法总比问题多，犯了错误要勇于承认，知错能改善莫大焉。在通往理想的道路上总会有踽踽独行的时候，勇者才能无惧。

再次，扎实地做好每件事。你说读小学了不高兴，因为没时间玩了，但是我最怕的是你为了玩而做事情不认真。你从小大概遗传了你母亲的利索，做事情很快。但是从今天开始我希望你做事情不仅是快，还要做踏实。人生苦短，一个人一生能得到的东西、做成的事不多，但是这个过程中虚幻的东西会不断地引诱你，所以不要心存侥幸，不要贪图捷径，努力做你想要的最好。你可以是工人，可以是农民，但是不管做什么你都要热情地做。

木木，在你上学前再叫一次你的小名。从此之后，你的肩上将会有一种叫做责任的东西，我希望你身上的独立、勇敢、踏实能够很好地负起各种责任。人生没有起跑线，路就在你脚下。你人生的第一次起步曾吓了你一跳，让你赶紧蹲下，但是这一次你要自信地走好。

google voice账号申请

发表于 2013-08-28 更新于 2014-04-01 分类于 随便说说 阅读次数： Valine：
本文字数： 741 阅读时长 ≈ 1 分钟

google voice账号可以免费打美国与加拿大的电话，而且据说google要开放它，这样的话能够用很少的话费全球通话了，所以尝试着申请一个玩玩还不错，特别是还没有美国号码同时又要经常跟本地人通话的同学。
在国内申请google voice至少有两个问题，一个就是美国地址，另一个就是美国的号码。在美国第一个问题可以忽略，所以唯一的问题就是获得一个美国号码。还好很多地方都提供了美国虚拟号码，我是用了http://www.virtualphoneline.com/signup/index.php，申请了一个号码，然后把它的电话转接到skype上。（这其中的过程有点复杂，但是忘了截图了，主要是先登录，然后free try a number，并在my number下面的Change forwarding all number中设置转接，这样就可以把google voice的验证电话转接到你可以接到的电话中了，你可以选择google talk，skype，msn还有其他你手头有的联系方式，关键是在接到google voice电话的时候有一个地方可以把验证号码输进去）
申请到了美国的虚拟号码，并做了相应的设置以后就可以登录google voice，在里面可以申请号码了，在弹出的窗口中输入你的虚拟号码然后就开始拨号，正常的话你能够通过你设置的方式接到电话，然后按来电语音提示输入google voice提供的验证号码（在google voice拨号的地方给出的两个数字）。
通过验证后，它会让你选择你所在的区号，或者电话号码的一些特征，也就是说电话号码可以自己挑一下，我最后挑了（302）9908209，所以你们如果要找我的话可以call这个号码，他有语音信箱，还有我把它也转接到我的skype、gtalk上了。

同学会

发表于 2013-08-05 更新于 2014-04-01 分类于 随便说说 阅读次数： Valine：
本文字数： 635 阅读时长 ≈ 1 分钟

暑假的中间阶段是每年同学会的最多的时候，99级的同学会也在这几天，到的同学不多，比我预想的还少。当然我也没去数数，到的人除了一个女同学确实没印象，其他都还算熟悉，基本上还能叫出名字。想工作以来，97、98级只是比我低一两届，当时也没叫他们叫我老师，虽然也教过他们课程，但是从现在想的起来的看，99级的学生可能更熟悉点，如果没记错的话他们有3门课的成绩是我给的。

这两年同学会开下来，98级的没参加，99这次是最少的，但是普遍的都不多，都是一半多一点，大家普遍热情不是很高。我自己在筹备班级的同学会的时候也有这种感觉，同学之间小圈子很多，平时交流也还可以，但是到一个班的聚会则有点勉强，很多人会用临时的一些小事来推脱。可能十年的时间还没给大家足够的考虑来参加同学会。实际上我看了20年以后的同学会普遍参与的人就会比较多了，一方面家庭的事务少了，事业上该成功的的也成功了，不成功的心境也恬淡了，另一方面也算是静极思动吧。

十年的同学会不景气可能也跟现在的社会风气有关，大家喜欢比较，特别是爱比钱。本来，大学同学是人生中很难得的一种情谊，大家差不多在人生最好的时光聚在一起，干了不少傻事糗事，也曾经意气风发过。聚会一方面也是增进友情，另一方面也给自己一个放松的机会。当然现在的同学会功利的东西多了点，其实是社会的风气使然，但是也让一些人对其产生反感，对有些人来说，同学会的时候有更重要的事情去做，所以也就无所谓来不来了。

明年我的第一届学生（当过两年班主任）毕业十年，不知道会怎样。
