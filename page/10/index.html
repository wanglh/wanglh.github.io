技术理性

发表于 2014-11-07 更新于 2017-03-06 分类于 理论研究 阅读次数： Valine：
本文字数： 198 阅读时长 ≈ 1 分钟

技术理性是哈贝马斯的概念。他根据人与自然的关系以及人与人的关系的两分，把劳动与交往区分开来。决定前者的就是技术理性，这种理性作用下使人与人之间的交往关系不能建立，而决定交往的关系的是交往理性。
技术理性只是体现了人的认知兴趣，但不考虑人类群体成员之间的价值观吗，所以技术是中性的。当前科学技术越界到处理人与人之间的关系，因而造成了各种社会问题，我们要约束它，使其在劳动的范围内起作用就可以解决问题。

win8使用投影仪时没声音

发表于 2014-10-29 分类于 实践探索 阅读次数： Valine：
本文字数： 184 阅读时长 ≈ 1 分钟

今天给石嘴山青年教师培训的时候发现我的机器没有声音输出，这个是非常奇怪的，因为培训前我还用过有声音的，而且这已经是第2次碰到了，所以想办法解决了一下。

实际上主要是现在的投影有些是通过hdmi来转接的，想我的机器就这样，而win8当用hdmi来投影的时候他的声音输出就是通过hdmi来传输的。所以当碰到这种情况的时候就要修改“播放设备”，改成原来的“喇叭/耳机”就行了。

公共领域

发表于 2014-10-26 更新于 2017-03-06 分类于 理论研究 阅读次数： Valine：
本文字数： 585 阅读时长 ≈ 1 分钟

阿多诺批判文化工业是资本主义社会对大众意识形态的控制工具，同时由于其商业化的特征导致了其服从于商品交换逻辑。

哈贝马斯在《公共领域的结构变迁》中提出了从封建社会向资本主义社会转变的过程中出现了公共权威与市民社会。公共权威指的是国家、司法手段等，而市民社会则是在公共权威保护下的私有经济领域。后者不但发展了经济关系，也促生了脱离经济活动的人际关系。这是一群资产阶级的文人雅士，他们以理性论证的方式对国家和权威进行争论与批判。在17世纪末，18世纪初，巴黎与伦敦一些由贵族聚会转变而来的沙龙和咖啡厅就是公共领域的雏形，也是现代知识分子活动的基本空间。这个空间非常重要，知识分子可以在这里通过理性论争的方式形成一种公共见解，也就是说个人见解可以在这里上升成为舆论，并且这种舆论独立于文化产业。

但是在资本主义社会由于公共权威的介入，大众传媒的商业化以及舆论管理技术的发展，这个公共领域名存实亡，逐渐蜕变为一个文化消费的地方。报纸由于商业化其内容必然会走向非政治化、个人化和煽情，而舆论管理技术的发展强调个人是私人公民而非消费者。于是公众成了一种被管理的资源，他们被从公共讨论和决策过程中被驱逐出去，而一些利益集团的主张又被合法化。

布迪厄的《关于电视》实际上就是接着哈贝马斯，从上述两个方面对电视在资本主义社会中的揭露：反民主的象征暴力与受商业逻辑制约的他律性。

moodle的分组技巧

发表于 2014-10-12 分类于 技术学习 阅读次数： Valine：
本文字数： 347 阅读时长 ≈ 1 分钟

moodle上开课，当碰到有多个平行班的时候，为了方便管理，我们都把每个班组成一个小组，这样在批改作业与算成绩的时候就比较好，不会搞乱。这里的关键就是使用自助选课，其他的选课方式有：人工选课是方便教师添加学生，访客可访问是指访客可以直接访问课程。

但是自助选课需要注意设置，首先自助选课的密码设为一个复杂点的密码，因为这里的关键是打开自助选课的选项，但是并不让学生通过这个自助选课的密码来选课。在自助选课设置中还有一个“使用分组选课密码”的选项，把它设为“是”，这样我们就可以在自助选课中通过使用分组密码来直接分组了。

所以自助选课设置好了后还需要去创立小组，同时为每个创立的小组设置一个“选课密码”。这样在不同的班里我们通过让学生使用不同的小组密码选课进来，在moodle中就会进入到不同的小组。

win8上用hyper-v虚拟个ubuntu玩玩

发表于 2014-09-19 更新于 2015-02-01 分类于 技术学习 阅读次数： Valine：
本文字数： 216 阅读时长 ≈ 1 分钟

用的东西比较早，有些需要windows，有些需要ubuntu，所以看到win8提供了hyper-v，就想着虚拟一个ubuntu玩玩。

虚拟本身比较简单，下载一个ubuntu的iso文件，然后创建虚拟的交换机，建议用网卡虚拟一个外部的，这样你的虚拟机也可以被外部访问。这里用无线网卡似乎不能虚拟出外部的，只能用有线网卡

然后就创建虚拟机，一路下去应该无碍。第一次启动可能会有点问题，你可以关闭虚拟机后再启动应该就好了。

然后你就尽情滴玩吧。
