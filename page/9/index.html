健康是福

发表于 2015-02-17 分类于 随便说说 阅读次数： Valine：
本文字数： 340 阅读时长 ≈ 1 分钟

现代人生活好了，越来越关注自身的健康问题。君不见各种社交媒体中，特别是最近流行的微信中不是心灵鸡汤，就是养生秘笈，这些东西的流行至少说明大家对健康的关注。

我大概爹娘给我打的基础比较好，一直以来很少生病，所以以前很少为健康发愁。但是也有那么几次，对健康也是几近渴求。

在广州读硕士的时候有一次腰椎间盘突出，每每半夜痛醒，夜不能寐找小说看，最后找到了《相约星期二》居然就一口气看完，虽然依然痛，但是第一次感觉健康的珍贵，真心觉得缺什么也不要缺健康。

第二次是一不小心一个脚趾骨折，从没有没这么受伤过，所以当时也是很忧伤，极度怀念身体健康的时候。

最近也有两次，睡下去挺好的人，醒来的时候天旋地转，虽然不是痛，但是这种难受的感觉我宁愿痛。所以，又开始怀念健康了。

keep fit，真的太要紧了。

各种短视

发表于 2015-02-07 分类于 随便说说 阅读次数： Valine：
本文字数： 512 阅读时长 ≈ 1 分钟

小刚导演说综艺电影是电影行业的自杀，还是有点道理的。不过商业社会各种存在都是合理的，关键是我们的公众需要一定的辨别力。拿研究来说，有些人三五天就写成一篇文章，有些人要一年半载，两者有不同的价值所在，前者多半是普及知识用的，而后者则是专业的知识创造，关键是要能区分这两篇文章背后所蕴含的学术意义，学术圈里实际上还是有自己的判断的，可惜的是现在管理学术圈的不一定懂学术，这也造成了数字学术，因为一个人学术再不济，能当领导数字还是看得懂的，数字多的就是好的。

电影业的问题也类似，因为电影本身就是大众货，要雅俗共赏，所以观众的鉴赏力很重要，可惜的是现在的大众传媒鉴赏口味都是从娃娃抓起的，看着儿子在追《奔跑吧，兄弟》我也偶尔陪他看一看。看着那些所谓的明星忸怩作态，矫揉造作我是觉得恶心的，设计的活动与情节也比较动物化。不过听说有人看这剧笑炸了肺，我想想还是让儿子看看吧，以免长大后笑点太低，也重蹈覆辙。所以，其实很难阻挡这类综艺电影的发展的，有点知识有点文化的人其实不用理会这些“黑”与“蜜”，做好自己最重要。想想我看的电影很少，基本上不是武侠就是香港的娱乐片，但这些经历似乎没有影响我做严肃的思考，而且我觉得自己是不是太严肃了点。

apache+passenger中配置redmine的注意事项

发表于 2015-01-21 更新于 2015-02-01 分类于 技术学习 阅读次数： Valine：
本文字数： 420 阅读时长 ≈ 1 分钟

重新安装了ubuntu的server，所以moodle，redmine要重新配置。

安装就不说了，轻车熟路了都比较简单。

今天碰到一个新问题，就是用apache+passenger发布redmine的时候，发现当redmine配置成子目录的时候没问题，但是当把redmine配置成独立站点的时候就老是403错误。

查了很多文档，总算在这里找到原因，原来passenger解析ror程序的时候，在apache里面配置的时候DocumentRoot** 必须是真正的public目录**，而不是我们经常做的软连接后的目录。

redmine能够使用，但是在访问版本库的时候老是出现500错误，这个时候要检查一下redmine目录的属主是否为www-data，或者执行一下：

sudo chown -R www-data:www-data /usr/share/redmine/

sudo chown -R www-data:www-data /var/www/redmine/

老师，你是不是不会做啊？

发表于 2014-12-20 更新于 2015-02-01 分类于 随便说说 阅读次数： Valine：
本文字数： 564 阅读时长 ≈ 1 分钟

喝了点酒睡觉，3点不到就醒来，而且还睡不着，所以早上去上课没什么精神，脑子转不动。碰巧今天上课示范的机器有点问题，想要做的效果出不来，在操作的时候也很机械。有些心急的学生在那里提醒我怎么做，但是还是做不出来。于是有个学生在那里说，老师，你是不是不会做啊？

说实在，我当时确实不会做，头脑已经停止思考了。我抬起头来看到一位同学用手臂顶了一下边上的那位同学，估计就是刚才给我指出问题的人。

说实在，我并不怕学生说老师，你怎么怎么，但是有点害怕师生之间缺乏相互信任。现在的大学教学是严重异化的，下面仅从技术教学的角度谈谈这个问题。第一种情况是教师只讲自己熟悉的东西，甚至二十年如一日，这个对人文学科大概问题不大，但是对于技术教学却是致命的，这个也可以从学生对教师的不信任看出来。第二种情况，当前大学也存在教师只讲学生听得懂的东西，这个本身也没问题，老师讲学生听不懂的东西终究违背了教学的宗旨，但是对学生的学习能力培养来说就有点问题了，经过老师一讲学生就没问题了，这种教学应该只存在基础教育，对大学教育这个不应该。我是倾向于教学生不容易自学的内容，这种内容对自己来说也是挑战，但是也会存在问题，有时候备课不充分，讲课效果不好，如果学生学习主动性缺失，效果也不好。

不管怎样，我还是会继续我的这个主张，接下去就是希望有更多的时间投入到教学中去。

关于教学

发表于 2014-12-04 分类于 随便说说 阅读次数： Valine：
本文字数： 476 阅读时长 ≈ 1 分钟

斯金纳说我们所做的任何事情和我们本身都是由奖励和惩罚的历史决定的，这句话也许有点偏颇，但是在经济学中有个词叫“路径依赖”说的就是人的行为依赖于他过去的所有行为，所以，学生特别是到了高等教育阶段的学生，他们已经在自己的生活中经历了太多不一样的事件，所以每个人很难得到一样的兴趣。说到底，高等教育中研究怎么让学生有兴趣远比如何让学生听得懂更重要。这是我的一家之言，但是这与我的教学历程是契合的。

我上课语气总是保持理性与平淡，所以话语缺乏煽动性，我不喜欢去说服学生，总是以一种愿者上钩的姿态来引导学生。我上技能类的课总是粗线条地勾勒各种技能的关键要点，细节的问题只有在学生问的时候才作解释。我上课的案例不是从某本书里抄袭来的，即使有所模仿但必然加入我自己的想法，我甚至都不喜欢用自己曾经讲过的例子，大千世界这么多地方都用得到这些技能我为什么要局限于一隅呢？一直认为，激发起学生的兴趣但又给他们的学习设置障碍，这样才能真正学好，把学生的问题都教没了，那必然导致遇到问题的时候手足无措。

我自己的学习过程就是这样的，对要学习的内容保持强烈的兴趣，同时克服各种困难。

技术理性

发表于 2014-11-07 更新于 2017-03-06 分类于 理论研究 阅读次数： Valine：
本文字数： 198 阅读时长 ≈ 1 分钟

技术理性是哈贝马斯的概念。他根据人与自然的关系以及人与人的关系的两分，把劳动与交往区分开来。决定前者的就是技术理性，这种理性作用下使人与人之间的交往关系不能建立，而决定交往的关系的是交往理性。
技术理性只是体现了人的认知兴趣，但不考虑人类群体成员之间的价值观吗，所以技术是中性的。当前科学技术越界到处理人与人之间的关系，因而造成了各种社会问题，我们要约束它，使其在劳动的范围内起作用就可以解决问题。
