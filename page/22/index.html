中秋节

发表于 2012-09-30 阅读次数： Valine：
本文字数： 442 阅读时长 ≈ 1 分钟

今天中秋节，成家后，特别是有小孩后第一次正儿八经地在家里过节。
早上自然醒（醒来不容易，昨天晚上睡不着，看电视到凌晨3点多），老婆建议去城里吃早餐，顺便去逛街，为了营造和谐的节日气氛，最终还是听老婆的建议。
肯德基已经过了早餐时间，老婆孩子就吃肯德基，对我来说除了早餐还有点兴趣，其他的食品就是垃圾，所以想想就去附近的五福园吃早饭了。吃完早饭已经过了10点了
逛街消费若干，到了午饭时间了，本想吃完了回去，但是木木说要回去。于是转到超市买些菜，想想这样也好，过节在家自己烧菜做饭，这样才像个家。说实在以前过节都是跑来跑去在父母家里，今天第一次三口之家自己过节还真是有点象征意义。毛蟹、菌菇煲、烤鸭还有些青菜，一家三口也就差不多了。吃好午饭2点半了。
午饭后休息，虽然一大堆的事情，今天铁定心不干了，电脑都不打开。5点左右考虑到木木看电视时间长了点就骑自行车带他出去一起去买了点水果，回来后玩那个小桌球，晚饭吃完快7点了，然后打电话给父母兄弟姐妹们。然后中国好声音差不多开始了，这个可以看一下。

木木与自行车

发表于 2012-09-04 更新于 2017-03-06 分类于 家庭生活 阅读次数： Valine：
本文字数： 183 阅读时长 ≈ 1 分钟

长得这么大还没有坐过自行车，所以拿到车那天去车行安装，木木就要跟着我去，说要坐回来。
第一次坐自行车，坐到后座上后悄悄地跟我说，爸爸，开慢点。
第一次坐自行车回到家说觉得pp有点疼。
以后每天晚上吃饭后没事就骑着自行车出去溜溜，木木也喜欢跟上，说现在坐自行车pp不疼了，不知道是不是真的。
每次骑着自行车带木木出去总有点怀旧的感觉，但对于坐惯了汽车的木木来说，这个好新鲜啊。
