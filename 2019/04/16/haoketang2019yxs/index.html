2019浙师大附属学校好课堂预选赛

发表于 2019-04-16 更新于 2019-05-31 分类于 随便说说 阅读次数： Valine：
本文字数： 310 阅读时长 ≈ 1 分钟

今年因为事先已经定好了最终去参赛的科目，所以预选赛组织比较简单，每个学校2个名额，仅报最终的参赛学科，不过最后还是有18位老师报名，初中7位，小学11位。

初中组今年印象最为深刻的是白云的陈进老师的课，老师在课堂上不断地通过问题引导学生探索，同时利用自制的实验工具，一步步达到最终的教学目标，整个过程学生都被吸引到老师提出的问题上，以及由此而来的实验装置的一步步改进中，教学效果非常好。

小学组没有现场听课，不过听说东阳花园今年有个语文老师上出一堂课，比较典型的利用技术弥补教师的不足，这也是很值得期待的一堂课。

每年好课堂前，总是忙忙碌碌，今年第四届了，感觉应该静下心来看看课堂的创新了，目前来看做得够多了，但是思考的不够。

# 智慧好课堂 # 浙江师大 # 附属学校
